## ما هذا التطبيق؟

هذا تطبيق ويب حديث يوفر:
- **مستشار الدعم النفسي بالذكاء الاصطناعي**: دردشة ثنائية اللغة توفر الدعم النفسي
- **تطبيق ويب تقدمي (PWA)**: يمكن تثبيته على هاتفك والعمل بدون إنترنت
- **واجهة احترافية**: تصميم حديث وسهل الاستخدام

## البدء السريع

### 1. المتطلبات
- Node.js 16 أو أحدث
- npm أو pnpm

### 2. التثبيت

```bash
# استنساخ أو تحميل المشروع
cd mnnnbvv

# تثبيت الحزم
pnpm install

# أو استخدام npm
npm install --legacy-peer-deps
```

### 3. إعداد البيئة

أنشئ ملف `.env.local` في جذر المشروع:

```env
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

للحصول على هذه المفاتيح:
1. اذهب إلى https://app.supabase.com
2. أنشئ مشروعاً جديداً أو استخدم مشروعاً موجوداً
3. انسخ المفاتيح من إعدادات المشروع

### 4. إعداد الذكاء الاصطناعي (Groq)

الخطوة الأهم لتشغيل الدردشة:

```bash
# احصل على مفتاح Groq من https://console.groq.com
# ثم أضفه في Supabase
supabase secrets set GROQ_API_KEY=your_groq_api_key
```

### 5. تشغيل المشروع

```bash
pnpm dev
```

ثم افتح المتصفح على: `http://localhost:5173`

## المميزات الرئيسية

### 🤖 مستشار الدعم النفسي
- دردشة ذكية تستجيب بطريقة متعاطفة وداعمة
- دعم لغتين: العربية والإنجليزية
- حفظ السجل في قاعدة البيانات

### 📱 تطبيق ويب تقدمي (PWA)
- تثبيت على الهاتف مثل أي تطبيق عادي
- استخدام بدون الإنترنت
- أيقونة على شاشة المنزل

### ✨ واجهة حديثة
- تصميم احترافي باستخدام Tailwind CSS
- تأثيرات سلسة مع Framer Motion
- تجربة مستخدم جميلة وسهلة

## البنية التقنية

المشروع مبني بطريقة معيارية (Modular Architecture):

```
src/
├── components/      # مكونات React
├── hooks/          # Hooks مخصصة
├── services/       # خدمات الذكاء الاصطناعي
├── contexts/       # Context API
└── integrations/   # تكاملات خارجية

supabase/
├── functions/      # Edge Functions (Groq AI)
└── migrations/     # نصوص قاعدة البيانات
```

## استكشاف الأخطاء

### مشكلة: الدردشة تظهر خطأ

**الحلول**:
1. تحقق من أن `GROQ_API_KEY` موضوع في Supabase secrets
2. افتح console (F12) وتحقق من الأخطاء
3. تأكد من أن متغيرات البيئة صحيحة

### مشكلة: زر تثبيت PWA لا يظهر

**الحلول**:
1. تأكد من استخدامك HTTPS (في production)
2. استخدم Chrome أو Edge (الدعم الأفضل لـ PWA)
3. جرب في browsing mode جديد

### مشكلة: خطأ في تثبيت الحزم

```bash
# استخدم هذا الأمر
npm install --legacy-peer-deps
```
